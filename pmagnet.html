pmagnet.core.core_loss(material, freq, flux, duty, algorithm='iGSE')¶

Calculate magnetic core loss of a material

Parameters

  • material – One of N27/N87/N49

  • freq – Frequency of excitation waveform (Hz)

  • flux – Relative Flux Density (mT) in a single waveform cycle, as an ndarray

  • duty – Duty (%) in a single waveform cycle, as an ndarray

  • algorithm – Only ‘iGSE’ supported for now

Returns

Core loss (kW/m^3)

pmagnet.core.core_loss_iGSE(freq, flux, duty, k_i, alpha, beta, n_interval=10001)¶

Calculate magnetic core loss using iGSE

Parameters

  • freq – Frequency of excitation waveform (Hz)

  • flux – Relative Flux Density (mT) in a single waveform cycle, as an ndarray

  • duty – Duty (%) in a single waveform cycle, as an ndarray

  • k_i – Stenmetz coefficient k_i

  • alpha – Steinmetz coefficient alpha

  • beta – Steinmetz coefficient beta

  • n_interval – No. of intervals to use to solve iGSE using trapezoidal rule

Returns

Core loss (kW/m^3)
